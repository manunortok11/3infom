Versionamento

O git é um software que permite realizar versões de um projeto. O github desktop é um software que faz a ligação entre vscode, git e github.

O github desktop permite fazer a sincronia entre o repositório local e o remoto

Commit

É o processo de confirmação de uma versão, ou seja, a cada operação é possivel confirmar a operação.

Push

Esse comando envia(sincroniza) a alterações realizadas no repositório local com o repositório remoto no github.
